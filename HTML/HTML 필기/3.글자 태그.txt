@ 글자 태그
-> 글자를 표현하기 위해 사용하는 태그

## h 태그 ##
<h?> </h?>
-> 일반적으로 제목을 입력 할때 사용하는 태그
-> 글자 크기를 조정할 때 사용하는 태그
-> h뒤에 숫자로 구분함

작성법
<h?> 제목 내용 </h?>

h1> h2 > h3 > h4 > h5 > h6

큰 글씨	         작은 글씨

## <br> 태그 ##
-> 문장의 줄 개행을 할때 사용

## <hr> 태그 ##
-> 페이지에 가로 줄을 만들어 줄 때 사용

## <p> 태그 ##
-> 한개의 단락을 만들 때 사용
-> 자동으로 문장 개행 후 한 줄 공백을 만들어 줌

-> &nbsp 띄어쓰기.

## <pre> 태그 ## 
-> 입력한 그대로 출력할 때 사용
-> 띄어쓰기, 들여쓰기, 줄 바굼이 입력된 그대로 출력 됨


## <strong> 태그 또는 <b> 태그 ##
-> 문장에서 문자를 강조하여 굵게 표시할 경우에 사용

## <em> 태그 또는 <i>  태그
-> 문자에 기울임을 주는 태그

## <mark> 태그 ##
-> 배경 부분이 노랑색이 되는 태그
-> 형광펜 표시가 된듯하게 출력

## <u> 태그 ##
-> 수평줄을 표시
-> <hr> 태그는 페이지 전체에 수평으로 줄을 표시하지만
<u> 태그는 해당 범위에만 수평줄을 표시


## <sub> 태그 ##
-> 아래 첨자 태그

## <sup> 태그 ##
-> 윗첨자 태그

## <s> 태그 ##
-> 취소선 태그

## <addr> 태그 ## 
-> 약자표시와 함께 마우스가 문자에 있으면 출력( Tooltip)

## <cite> 태그 ##
-> 웹 문서나 포스트에서 참조 자료를 표시할때0 사용하는 태그
-> 이탤릭으로 표시 (브라우저마다 상이함)
-> 기울임과 큰 차이는 없음 (다만 참조 사이트 명시 했는지를 찾아 볼 수 있음)

## <blockquote> 태그 ##
-> 다른 블로그나 사이트의 글을 인용할 경우 사용하는 태그
-> 자동 들여쓰기가 되어 다른 텍스트와 구별할 수 있음

## <q> 태그  ##
-> blockquote 태그와 비슷하게 다른 블로그나 사이트의 글을 인용할 경우에 사용하는 태그
-> 자동 들여쓰기가 되고 인용문구에 "" 표시가 됨
