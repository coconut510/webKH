## <a> 태그 ##

-> 페이지에서 해당 부분을 클릭하면 지정된 페이지로 이동하는 태그
-> 외부 사이트 연결, 문서 내부에서 이동 가능함

속성
href		: 링크한 페이지의 id 값이나 사이트 주소 지정
target		: 링크 페이지가 표시될 위치(새창, 현재창) 지정 가능
		새창 	: _blank
		현재창 	: _self
ex)
<a href="https://www.naver.com">네이버</a>
